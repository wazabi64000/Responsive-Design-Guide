Introduction

Les interactions doivent s’adapter aux différents écrans : menus, sliders, boutons, tableaux et formulaires doivent rester utilisables et accessibles. Ce chapitre montre comment créer des éléments interactifs flexibles et adaptatifs.

Difficultés courantes

  • Menus non utilisables sur mobile.
  • Sliders ou carrousels non adaptatifs.
  • Formulaires trop larges ou éléments qui débordent.
  • Éléments interactifs difficiles à toucher sur petit écran.

Solutions et bonnes pratiques

  • Menus hamburger ou collapsibles pour petits écrans.
  • Sliders et carrousels adaptatifs avec largeur flexible et scroll horizontal.
  • Formulaires empilés sur mobile et en colonnes sur grand écran.
  • JavaScript pour ajuster comportements selon la largeur de l’écran.
  • Feedback visuel clair pour toutes les interactions.

Exemples de code

Menu hamburger simple

<button id="menu-btn">☰ Menu</button>
<nav id="main-menu">
  <ul>
    <li><a href="#">Accueil</a></li>
    <li><a href="#">Services</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

<script>
const btn = document.getElementById('menu-btn');
const menu = document.getElementById('main-menu');
btn.addEventListener('click', () => menu.classList.toggle('open'));
</script>

Slider adaptatif

.slider {
  display: flex;
  overflow-x: auto;
  scroll-snap-type: x mandatory;
}
.slide {
  flex: 0 0 80%;
  margin-right: 1rem;
  scroll-snap-align: start;
}

Formulaire responsive

@media (max-width: 768px) {
  form {
    display: flex;
    flex-direction: column;
  }
  form input, form button {
    width: 100%;
    margin-bottom: 1rem;
  }
}

Outils et ressources

  • Chrome DevTools – mode responsive et simulation d’interactions tactiles
  • Responsinator – tester menus et sliders sur mobiles
  • Can I Use – compatibilité JS et CSS pour interactions
  • Swiper.js ou Slick Carousel – sliders adaptatifs et accessibles

Conclusion

En adaptant les interactions aux différentes tailles d’écran et aux dispositifs tactiles, vous améliorez l’expérience utilisateur pour tous. Les menus, sliders et formulaires doivent être flexibles, intuitifs et accessibles.
