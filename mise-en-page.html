Introduction

La mise en page responsive permet à vos contenus de s'adapter automatiquement à toutes les tailles d’écran grâce à Flexbox et CSS Grid. Elle garantit un affichage harmonieux et lisible sur mobile, tablette et desktop.

Techniques et bonnes pratiques

  • Utiliser display: flex; et flex-wrap: wrap; pour des conteneurs flexibles.
  • CSS Grid pour des layouts complexes mais adaptatifs avec repeat(auto-fit, minmax(...)).
  • Unités relatives (%, em, rem, vw) plutôt que fixes.
  • Media queries pour ajuster colonnes et marges selon la largeur de l’écran.

Exemples améliorés

Flexbox responsive

<div class="flex-container">
  <div class="flex-item">1</div>
  <div class="flex-item">2</div>
  <div class="flex-item">3</div>
</div>

<style>
.flex-container {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}
.flex-item {
  flex: 1 1 200px;
  background: #bb86fc;
  color: #121212;
  padding: 1rem;
  text-align: center;
  border-radius: 8px;
}
@media (max-width: 768px) {
  .flex-item { flex: 1 1 100%; }
}
</style>

CSS Grid responsive

.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 1rem;
}
.grid-item {
  background: #03dac6;
  color: #121212;
  padding: 1rem;
  text-align: center;
  border-radius: 8px;
}
