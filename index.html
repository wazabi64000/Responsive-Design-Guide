Introduction

Le Responsive Design permet à vos sites web de s’adapter automatiquement à tous les appareils, garantissant ainsi une expérience utilisateur optimale sur mobile, tablette et desktop.

Ce guide vous fournit des techniques modernes, des exemples concrets et des ressources utiles pour maîtriser le design adaptatif.

Ressources utiles

  • MDN – Responsive Design
  • CSS Tricks – Media Queries
  • Can I Use – Compatibilité navigateurs
  • Responsinator – Tester mobile

Exemple simple HTML/CSS responsive

<div class="container">
  <div class="item">Bloc 1</div>
  <div class="item">Bloc 2</div>
  <div class="item">Bloc 3</div>
</div>

<style>
.container {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}
.item {
  flex: 1 1 250px;
  background: #bb86fc;
  color: #121212;
  padding: 1rem;
  border-radius: 8px;
  text-align: center;
}
@media (max-width: 768px) {
  .item { flex: 1 1 100%; }
}
</style>
