Introduction

Une typographie responsive s'adapte à toutes les tailles d'écran pour offrir une lecture confortable et hiérarchisée.

Exemples de code

Unités relatives

body {
  font-size: 1rem; /* 16px */
}
h1 { font-size: 2rem; } /* 32px */
p { font-size: 1rem; }

Media queries pour mobile

@media (max-width: 768px) {
  body { font-size: 0.9rem; }
  h1 { font-size: 1.5rem; }
}

Typographie fluide avec vw

h2 {
  font-size: 3vw; /* s'adapte à la largeur de l'écran */
}
