Introduction

Les médias doivent être flexibles pour ne pas déformer le layout et optimiser la vitesse de chargement. Utilisez srcset, <picture> et CSS responsive.

Exemples pratiques

Image responsive CSS

img {
  max-width: 100%;
  height: auto;
  border-radius: 8px;
}

Image avec srcset

<img src="small.jpg"
     srcset="small.jpg 480w, medium.jpg 768w, large.jpg 1200w"
     sizes="(max-width: 768px) 100vw, 50vw"
     alt="Exemple image responsive">

Vidéo responsive

.video-container {
  position: relative;
  padding-bottom: 56.25%; /* 16:9 */
  height: 0;
}
.video-container iframe {
  position: absolute;
  width: 100%;
  height: 100%;
  top:0;
  left:0;
}
